# SemPKM Marketing Screenshots

Auto-generated by Playwright from the live application with seed data.

## Regenerating

```bash
cd e2e
npm run env:start        # Start isolated Docker test stack
npm run env:wait         # Wait for healthy
npm run screenshots      # Capture all screenshots
npm run env:stop         # Tear down
```

Or run headed to watch the browser navigate:

```bash
npm run screenshots:headed
```
