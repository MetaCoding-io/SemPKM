<objective>
Add server-side auth dependencies to every unprotected HTML router and wire user provenance into browser write endpoints.

Purpose: Currently all browser, views, admin, shell, and debug HTML routes have zero server-side auth -- a curl request bypasses the client-side JS auth check entirely. This plan adds Depends(get_current_user) to read endpoints and Depends(require_role("owner", "member")) to write endpoints, using the existing auth dependencies from Phase 6. Browser write endpoints (create_object, save_object, save_body) also gain performed_by + performed_by_role provenance, matching the pattern already established in commands/router.py.

Output: All 31 HTML endpoints protected with server-side auth, 3 browser write endpoints recording user provenance
</objective>
