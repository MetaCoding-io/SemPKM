**`async def get_event_detail(self, event_iri: str) -> EventDetail | None`**:
1. Query data triples from the event graph (exclude event metadata triples):
```sparql
PREFIX sempkm: <urn:sempkm:>
SELECT ?s ?p ?o
WHERE {
  GRAPH <{event_iri}> {
    ?s ?p ?o .
    FILTER(
      ?s != <{event_iri}> ||
      ?p NOT IN (
        sempkm:timestamp, sempkm:operationType, sempkm:affectedIRI,
        sempkm:performedBy, sempkm:performedByRole, sempkm:description,
        <http://www.w3.org/1999/02/22-rdf-syntax-ns#type>
      )
    )
  }
}
```
Also query event metadata (to get timestamp and opType) with:
```sparql
PREFIX sempkm: <urn:sempkm:>
SELECT ?timestamp ?opType ?affectedIRI ?performedBy ?description
WHERE {
  GRAPH <{event_iri}> {
    <{event_iri}> sempkm:timestamp ?timestamp ;
                  sempkm:operationType ?opType .
    OPTIONAL { <{event_iri}> sempkm:affectedIRI ?affectedIRI }
    OPTIONAL { <{event_iri}> sempkm:performedBy ?performedBy }
    OPTIONAL { <{event_iri}> sempkm:description ?description }
  }
}
```
If no metadata rows returned: return None (event not found).

2. Build `new_values` dict: for each data triple where the subject is the primary affected IRI (first affectedIRI), map `predicate_iri -> object_value_string`.

3. Reconstruct `before_values`: For each predicate in `new_values`, find the most recent prior event that recorded a value for that subject+predicate using backward query:
```sparql
PREFIX sempkm: <urn:sempkm:>
PREFIX xsd: <http://www.w3.org/2001/XMLSchema#>
SELECT ?old_value
WHERE {
  GRAPH ?prev_event {
    ?prev_event sempkm:timestamp ?prev_ts .
    <{subject_iri}> <{predicate_iri}> ?old_value .
  }
  FILTER(STRSTARTS(STR(?prev_event), "urn:sempkm:event:"))
  FILTER(?prev_ts < "{this_event_timestamp}"^^xsd:dateTime)
}
ORDER BY DESC(?prev_ts)
LIMIT 1
```
Run this per-predicate. Wrap in try/except — if query fails or times out, `before_values[pred] = "(unknown)"`.

4. For `body.set` events: `new_values` will contain a triple with predicate for the body property. The `before_values` will contain the old body. Call `_compute_body_diff(old_body, new_body)` and store in `body_diff`. For non-body.set events, `body_diff = None`.

5. For `object.create` / `edge.create`: skip backward query (no before state). `before_values = {}`.

**`def _compute_body_diff(self, old_body: str, new_body: str) -> list[dict]`** (sync method):
```python
import difflib
def _compute_body_diff(self, old_body: str, new_body: str) -> list[dict]:
    old_lines = (old_body or "").splitlines(keepends=True)
    new_lines = (new_body or "").splitlines(keepends=True)
    diff_lines = []
    for line in difflib.unified_diff(old_lines, new_lines, lineterm=''):
        if line.startswith('+++') or line.startswith('---') or line.startswith('@@'):
            continue
        elif line.startswith('+'):
            diff_lines.append({'type': 'add', 'text': line[1:]})
        elif line.startswith('-'):
            diff_lines.append({'type': 'remove', 'text': line[1:]})
        else:
            diff_lines.append({'type': 'context', 'text': line[1:] if line.startswith(' ') else line})
    return diff_lines if diff_lines else [{'type': 'context', 'text': '(no changes)'}]
```

**`async def build_compensation(self, event_iri: str, detail: EventDetail) -> "Operation | None"`**:

```python
async def build_compensation(self, event_iri: str, detail: EventDetail) -> Operation | None:
    op_type = detail.summary.operation_type
    subject_iri = detail.summary.affected_iris[0] if detail.summary.affected_iris else None

    if op_type == "object.patch":
        if not detail.before_values or not subject_iri:
            return None  # Can't undo without before values
        # Build new object.patch with old values
        from rdflib import URIRef, Literal
        data_triples = []
        materialize_deletes = []
        for pred_iri, old_val in detail.before_values.items():
            if old_val == "(unknown)":
                continue
            # New value (to delete): from detail.new_values
            new_val = detail.new_values.get(pred_iri)
            if new_val:
                materialize_deletes.append((URIRef(subject_iri), URIRef(pred_iri), Literal(new_val)))
            data_triples.append((str(URIRef(subject_iri)), pred_iri, old_val))
        return Operation(
            operation_type="object.patch",
            affected_iris=[subject_iri],
            description=f"Undo object.patch: {event_iri}",
            data_triples=[(URIRef(s), URIRef(p), Literal(o)) for s, p, o in data_triples],
            materialize_inserts=[(URIRef(subject_iri), URIRef(pred_iri), Literal(old_val)) for pred_iri, old_val in detail.before_values.items() if old_val != "(unknown)"],
            materialize_deletes=materialize_deletes,
        )

    elif op_type == "body.set":
        if not detail.before_values or not subject_iri:
            return None
        # Find body predicate (the predicate whose value is the body content)
        # Check app/events/models.py or body_set.py for the body predicate IRI
        # It is likely the predicate that has the body content as its object in data_triples
        body_pred = None
        old_body = None
        for pred_iri, old_val in detail.before_values.items():
            body_pred = pred_iri
            old_body = old_val
            break
        if not body_pred or old_body is None:
            return None
        from rdflib import URIRef, Literal
        return Operation(
            operation_type="body.set",
            affected_iris=[subject_iri],
            description=f"Undo body.set: {event_iri}",
            data_triples=[(URIRef(subject_iri), URIRef(body_pred), Literal(old_body))],
            materialize_inserts=[(URIRef(subject_iri), URIRef(body_pred), Literal(old_body))],
            materialize_deletes=[(URIRef(subject_iri), URIRef(body_pred), detail.new_values.get(body_pred, Literal("")))],
        )

    elif op_type == "edge.create":
        if not detail.data_triples:
            return None
        # Delete the edge triples that were created
        from rdflib import URIRef, Literal
        materialize_deletes = []
        for s_str, p_str, o_str in detail.data_triples:
            try:
                s = URIRef(s_str)
                p = URIRef(p_str)
                o = URIRef(o_str) if o_str.startswith("http") or o_str.startswith("urn") else Literal(o_str)
                materialize_deletes.append((s, p, o))
            except Exception:
                pass
        return Operation(
            operation_type="edge.create.undo",
            affected_iris=detail.summary.affected_iris,
            description=f"Undo edge.create: {event_iri}",
            data_triples=[],
            materialize_inserts=[],
            materialize_deletes=materialize_deletes,
        )

    elif op_type == "edge.patch":
        # Same pattern as object.patch but for edge annotations
        if not detail.before_values or not subject_iri:
            return None
        from rdflib import URIRef, Literal
        return Operation(
            operation_type="edge.patch",
            affected_iris=detail.summary.affected_iris,
            description=f"Undo edge.patch: {event_iri}",
            data_triples=[(URIRef(subject_iri), URIRef(pred_iri), Literal(old_val)) for pred_iri, old_val in detail.before_values.items() if old_val != "(unknown)"],
            materialize_inserts=[(URIRef(subject_iri), URIRef(pred_iri), Literal(old_val)) for pred_iri, old_val in detail.before_values.items() if old_val != "(unknown)"],
            materialize_deletes=[(URIRef(subject_iri), URIRef(pred_iri), Literal(new_val)) for pred_iri, new_val in detail.new_values.items()],
        )

    return None  # Not reversible
```

@router.post("/events/{event_iri:path}/undo")
async def undo_event(
    request: Request,
    event_iri: str,
    user: User = Depends(require_role("owner", "member")),
    client: TriplestoreClient = Depends(get_triplestore_client),
):
    from app.events.query import EventQueryService
    from app.events.store import EventStore
    from urllib.parse import unquote
    from rdflib import URIRef
    decoded_iri = unquote(event_iri)
    query_svc = EventQueryService(client)
    detail = await query_svc.get_event_detail(decoded_iri)
    if not detail:
        return JSONResponse(status_code=404, content={"error": "Event not found"})
    compensation = await query_svc.build_compensation(decoded_iri, detail)
    if not compensation:
        return JSONResponse(status_code=400, content={"error": "This event cannot be undone"})
    event_store = EventStore(client)
    user_iri = URIRef(f"urn:sempkm:user:{user.id}")
    await event_store.commit([compensation], performed_by=user_iri, performed_by_role=user.role)
    return JSONResponse(content={"status": "ok", "message": "Undo applied successfully"})

**Add JS `sempkmUndoEvent` to `workspace.js`** (or inline in workspace.html):

Add to `workspace.js` as a window-exported function:
```javascript
window.sempkmUndoEvent = function(eventIri, btn) {
    if (!window.confirm('Undo this event? This will create a new compensating event.\n\nNote: Any changes made to the same fields after this event will also be reverted.')) {
        return;
    }
    btn.disabled = true;
    btn.textContent = 'Undoing…';
    fetch('/browser/events/' + encodeURIComponent(eventIri) + '/undo', {
        method: 'POST',
        headers: { 'X-CSRFToken': document.cookie.match(/csrftoken=([^;]+)/)?.[1] || '' }
    }).then(function(res) {
        if (res.ok) {
            // Reload the event log pane
            htmx.ajax('GET', '/browser/events', {
                target: '#panel-event-log',
                swap: 'innerHTML'
            });
        } else {
            res.json().then(function(d) {
                alert('Undo failed: ' + (d.error || 'Unknown error'));
                btn.disabled = false;
                btn.textContent = 'Undo';
            });
        }
    }).catch(function() {
        alert('Undo failed: network error');
        btn.disabled = false;
        btn.textContent = 'Undo';
    });
};
```

Check if the project uses CSRF tokens. Look at other `fetch` POST calls in `workspace.js` or `sidebar.js` for the existing pattern. If no CSRF is used (FastAPI typically doesn't require it for cookie-auth apps), remove the `X-CSRFToken` header.
