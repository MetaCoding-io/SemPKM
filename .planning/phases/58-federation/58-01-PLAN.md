         a. Accepts `graph_id` (UUID portion of `urn:sempkm:shared:{uuid}`) and `since` query param (ISO timestamp string).
         b. Constructs full graph IRI: `urn:sempkm:shared:{graph_id}`.
         c. Queries event graphs that: (1) have `sempkm:graphTarget` matching the shared graph IRI, (2) have timestamp > `since`, (3) do NOT have `sempkm:syncSource` matching the requesting instance (passed via query param `requester` or derived from HTTP Signature key_id -- for now use query param).

            ```sparql
            SELECT ?event ?timestamp ?data_triple
            WHERE {
              GRAPH ?event {
                ?event a sempkm:Event ;
                       sempkm:timestamp ?timestamp ;
                       sempkm:graphTarget <{graph_iri}> .
                FILTER(?timestamp > "{since}"^^xsd:dateTime)
                FILTER NOT EXISTS { ?event sempkm:syncSource <{requester_url}> }
                ?s ?p ?o .
                FILTER(?s != ?event)
              }
            }
            ORDER BY ?timestamp
            ```
