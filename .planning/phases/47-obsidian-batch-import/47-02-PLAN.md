**2. Create import_progress.html:**

Template structure:
- Step bar with `current_step=5` and an "Importing..." label
- Progress bar container with `id="import-progress-bar"` showing percentage
- Phase indicator text ("Importing objects..." / "Creating edges...")
- Counter text ("X of N")
- Scrolling log container (`id="import-log"`) with terminal-like styling (dark background, monospace font, max-height with overflow-y: auto, auto-scrolls to bottom)
- JavaScript at bottom that:
  - Creates `EventSource` connecting to `/browser/import/{{ import_id }}/execute/stream`
  - On `import_progress` event: update progress bar width, counter text, phase text, append log line with current_file
  - On `import_complete` event: close EventSource, fetch summary via `htmx.ajax('GET', '/browser/import/{{ import_id }}/summary', {target: '#import-area', swap: 'outerHTML'})`
  - On `import_error` event: close EventSource, show error message in the log area
  - Auto-scroll log div to bottom on each new line

**3. Create import_summary.html:**

Template structure:
- Step bar with `current_step=5` and "Import Complete" label
- Stat cards row (similar to scan results dashboard pattern):
  - "Created" (big number, success color)
  - "Edges" (big number)
  - "Skipped" (existing + errors combined, warning color if > 0)
  - "Duration" (formatted seconds)
- If `result.skipped_existing > 0`: info notice "N objects skipped (already imported)"
- If `result.errors`: expandable section "Errors (N)" with table of path + message
- If `result.unresolved_links`: expandable section "Unresolved Links (N)" with table of source note + target name. This should be useful per user request -- show both the source note that had the link and what it tried to link to.
- Action buttons:
  - "Browse Imported Objects" (`btn btn-primary`): `hx-get="/workspace"` with `hx-push-url="true"` targeting the body, navigating to the workspace. Also dispatch `sempkm:nav-refresh` custom event via inline JS to refresh the nav tree.
  - "Import More" (`btn btn-outline`): `hx-get="/browser/import"` to go back to upload
  - "Discard Vault" (`btn btn-outline btn-danger`): `hx-post="/browser/import/{{ import_id }}/discard"` to clean up
- Lucide icon initialization script
