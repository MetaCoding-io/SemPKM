  <files>frontend/static/js/workspace-layout.js</files>
  <action>
Add to `workspace-layout.js` these functions, called from `renderGroupTabBar()` after building each tab element:

**isDragging flag** (module-level var):
```javascript
var isDragging = false;
```

**initTabDrag(tabEl, tabId, groupId):**
- Sets `tabEl.setAttribute('draggable', 'true')`
- `dragstart`: Set isDragging = true; dataTransfer.setData('text/plain', JSON.stringify({tabId, sourceGroupId: groupId})); effectAllowed = 'move'; create ghost clone with opacity 0.7, position fixed top -1000px, append to body, setDragImage(ghost, 20, 10), setTimeout to remove ghost from body; tabEl.classList.add('dragging')
- `dragend`: tabEl.classList.remove('dragging'); document.querySelectorAll('.drop-indicator').forEach(el => el.classList.remove('drop-indicator-active')); document.querySelectorAll('.group-tab-bar').forEach(el => el.classList.remove('tab-bar-drag-over')); setTimeout(() => { isDragging = false; }, 0)
- In tab's click handler in renderGroupTabBar: add `if (isDragging) return;` guard before calling switchTab (per Pitfall 3 from research)

**initTabBarDropZone(tabBarEl, targetGroupId):**
- `dragover`: e.preventDefault(); e.dataTransfer.dropEffect = 'move'; tabBarEl.classList.add('tab-bar-drag-over'); _updateInsertionIndicator(tabBarEl, e.clientX)
- `dragleave`: only remove drag-over if relatedTarget is not inside tabBarEl (use `!tabBarEl.contains(e.relatedTarget)`); tabBarEl.classList.remove('tab-bar-drag-over'); hide drop indicator
- `drop`: e.preventDefault(); tabBarEl.classList.remove('tab-bar-drag-over'); hide indicator; parse data; find insertBeforeTabId from insertion position; call `layout.moveTab(data.tabId, data.sourceGroupId, targetGroupId, insertBeforeTabId)`; then trigger re-render

**_updateInsertionIndicator(tabBarEl, clientX):**
- Find or create `.drop-indicator` inside tabBarEl
- Iterate `.workspace-tab` elements; find first tab whose midX > clientX; place indicator before that tab (left = rect.left - tabBarEl.getBoundingClientRect().left + tabBarEl.scrollLeft)
- If none found, place at scrollWidth
- Add `.drop-indicator-active` class

**Right-edge drop zone** (attach to `#editor-groups-container`):
- `dragover`: check if e.clientX > containerRect.right - 80; if yes and layout.groups.length < 4, show right-edge indicator (semi-transparent overlay div `#right-edge-drop-zone` with 80px width, full height, positioned absolute right 0)
- `dragleave`: hide right-edge indicator if relatedTarget not in zone
- `drop`: if cursor in right 80px zone, parse data; layout.addGroup(); var newGroup = layout.groups[layout.groups.length-1]; layout.moveTab(data.tabId, data.sourceGroupId, newGroup.id, null); hide indicator; call recreateGroupSplit(layout)

**Update renderGroupTabBar()** to call `initTabDrag(tabEl, tab.id, group.id)` for each tab element and `initTabBarDropZone(tabBarEl, group.id)` once per tab bar.

**WorkspaceLayout.moveTab(tabId, sourceGroupId, targetGroupId, insertBeforeTabId):**
- Find source group and target group
- Remove tab from source group tabs array
- If insertBeforeTabId: insert before that tab in target; else push to end
- If source group becomes empty AND it's not the last group: removeGroup(sourceGroupId) (which triggers recreateGroupSplit)
- Else: save() and trigger partial re-renders of both groups' tab bars
- Load the moved tab content in the target group: call loadTabInGroup(targetGroupId, tabId)

**Add to workspace-layout.js — showTabContextMenu(e, tabId, groupId):**

Pattern from 14-RESEARCH.md Pattern 6 verbatim:
1. `e.preventDefault(); e.stopPropagation();`
2. Remove any existing `#tab-context-menu`
3. Build menu div with class `context-menu`, id `tab-context-menu`
4. Items array:
   - `{ label: 'Close', action: () => layout.removeTabFromGroup(tabId, groupId) }`
   - `{ label: 'Close Others', action: () => closeOtherTabsInGroup(tabId, groupId) }`
   - `{ label: '---' }` (separator)
   - `{ label: 'Split Right', action: () => { splitRight(groupId); } }`
   - If layout.groups.length > 1: build "Move to Group" items for each other group: `{ label: 'Move to Group ' + (idx+1), action: () => layout.moveTab(tabId, groupId, otherGroup.id, null) }`
5. Position: clamp to viewport (clientX/clientY with menuRect correction per research)
6. Append to document.body
7. Dismiss: setTimeout to add click-outside listener; also dismiss on Escape keydown

**closeOtherTabsInGroup(keepTabId, groupId) helper:**
- Get group; close all tabs except keepTabId; call removeTabFromGroup for each

**Wire context menu** in `renderGroupTabBar()`: each tab element gets `addEventListener('contextmenu', (e) => showTabContextMenu(e, tab.id, group.id))`
