Add to `workspace.js` (IIFE scope, before the exposed window.* exports):

**Panel state** (module-level):
```javascript
var PANEL_KEY = 'sempkm_bottom_panel';
var panelState = { open: false, height: 30, activeTab: 'sparql', maximized: false };
```

**restorePanelState():** reads PANEL_KEY from localStorage, merges into panelState with Object.assign.

**savePanelState():** localStorage.setItem(PANEL_KEY, JSON.stringify(panelState)).

**_applyPanelState():**
- Get `#bottom-panel`, `#panel-resize-handle`, `.editor-column` (or `#editor-pane` — use the actual class from Plan 01 output)
- If !panelState.open: panel.style.display = 'none'; handle.classList.remove('panel-open'); editorCol.classList.remove('panel-maximized')
- If panelState.maximized: panel.style.display = ''; handle.classList.add('panel-open'); editorCol.classList.add('panel-maximized'); panel.style.height = ''
- Else (open, not maximized): panel.style.display = ''; handle.classList.add('panel-open'); editorCol.classList.remove('panel-maximized'); var parentH = panel.parentElement.getBoundingClientRect().height; panel.style.height = (parentH * panelState.height / 100) + 'px' (per Pitfall 5 from research — always set pixels, not %)
- Set active panel tab: querySelectorAll('.panel-tab').forEach remove active; querySelector(`[data-panel="${panelState.activeTab}"]`).classList.add('active'); querySelectorAll('.panel-pane').forEach remove active; getElementById('panel-'+panelState.activeTab.replace('-','') ... use document.querySelector('#panel-' + panelState.activeTab).classList.add('active')
- Re-init Lucide icons in panel if open: if (typeof lucide !== 'undefined') lucide.createIcons({attrs: {class: ['lucide']}})

**toggleBottomPanel():**
- panelState.open = !panelState.open; panelState.maximized = false; _applyPanelState(); savePanelState()

**maximizeBottomPanel():**
- if (!panelState.open) panelState.open = true; panelState.maximized = !panelState.maximized; _applyPanelState(); savePanelState()

**initBottomPanelResize():**
Exact implementation from 14-RESEARCH.md Pattern 5:
- Get `#panel-resize-handle` and `#bottom-panel`; if either missing, return
- mousedown: record startY, startHeight via getBoundingClientRect(); set body userSelect none + cursor row-resize; add mousemove and mouseup listeners to document
- mousemove: delta = startY - e.clientY; workspaceH = panel.parentElement.getBoundingClientRect().height; newHeight = clamp(startHeight + delta, 80, workspaceH * 0.8); panel.style.height = newHeight + 'px'; panelState.height = (newHeight / workspaceH * 100)
- mouseup: body.style.userSelect = ''; body.style.cursor = ''; savePanelState(); remove listeners; e.preventDefault() on mousedown

**initPanelTabs():**
- document.querySelectorAll('.panel-tab').forEach: addEventListener click, on click: panelState.activeTab = btn.dataset.panel; savePanelState(); _applyPanelState()

**Wire maximize button:**
- In an initBottomPanel() function: document.getElementById('panel-maximize-btn')?.addEventListener('click', maximizeBottomPanel)
- initPanelTabs()
- initBottomPanelResize()
- restorePanelState(); _applyPanelState()
