**SessionStorage migration (MUST run first at init):**
```javascript
function migrateTabState() {
  if (sessionStorage.getItem('sempkm_workspace_layout')) return;
  var oldTabs = JSON.parse(sessionStorage.getItem('sempkm_open_tabs') || '[]');
  var oldActive = sessionStorage.getItem('sempkm_active_tab');
  var layout = {
    groups: [{ id: 'group-1', tabs: oldTabs, activeTabId: oldActive || (oldTabs.length > 0 ? oldTabs[0].iri : null), size: 100 }],
    activeGroupId: 'group-1'
  };
  sessionStorage.setItem('sempkm_workspace_layout', JSON.stringify(layout));
  sessionStorage.removeItem('sempkm_open_tabs');
  sessionStorage.removeItem('sempkm_active_tab');
}
```

**WorkspaceLayout class** (LAYOUT_KEY = 'sempkm_workspace_layout'):
- `addGroup()`: max 4 groups guard; generates id as 'group-' + Date.now(); redistributes equal sizes (100 / (groups.length + 1)); calls save(); triggers recreateGroupSplit(); returns new group id
- `removeGroup(groupId)`: guard if groups.length === 1 (no-op); splice from array; redistribute tabs to left neighbor (Math.max(0, idx-1)); equal sizes; update activeGroupId if removed; save(); recreateGroupSplit()
- `moveTab(tabId, sourceGroupId, targetGroupId, insertBeforeTabId)`: remove from source, insert into target at correct position; if source becomes empty, removeGroup(sourceGroupId); save(); partial re-render of both tab bars
- `addTabToGroup(tab, groupId)`: adds tab object to group; calls save() and renderGroupTabBar(groupId)
- `removeTabFromGroup(tabId, groupId)`: removes tab; updates activeTabId; if last tab and not last group, removeGroup(groupId); else save() and re-render; if last group last tab, show empty state
- `setActiveGroup(groupId)`: updates activeGroupId; adds .editor-group-active class to correct group container; save()
- `save()`: sessionStorage.setItem(LAYOUT_KEY, JSON.stringify({groups, activeGroupId}))
- `static restore()`: reads LAYOUT_KEY, returns WorkspaceLayout or null

**recreateGroupSplit(layout) function:**
1. Manually destroy any CodeMirror/Split.js instances in groups being removed: call window.runCleanup and window._sempkmSplits destroy (per Pitfall 8 in research)
2. Destroy groupSplitInstance if exists
3. Set editorPane.innerHTML = '' to clear all groups
4. Create group DOM for each group:
   ```html
   <div class="editor-group" id="{group.id}">
     <div class="group-tab-bar" id="tab-bar-{group.id}"></div>
     <div class="group-editor-area" id="editor-area-{group.id}"></div>
   </div>
   ```
5. If groups.length > 1: create Split.js with gutterSize: 1, gutterClass: 'gutter-editor-groups', minSize: 200, cursor: 'col-resize', onDragEnd saves sizes to layout
6. For each group: call renderGroupTabBar(group) and if group.activeTabId call loadTabInGroup(group.id, group.activeTabId)

**renderGroupTabBar(group) function:** builds tab HTML matching existing .workspace-tab/.tab-close pattern from workspace.js renderTabs(). Each tab has draggable="true" (stub for Plan 02 DnD hookup). Add `data-tab-id` and `data-group-id` attributes. Add contextmenu handler stub (Plan 02).

**loadTabInGroup(groupId, tabId) function:** uses htmx.ajax to load content into `#editor-area-{groupId}`. Detects if tab.isView (view: prefix) or object IRI. Mirrors existing _loadTabContent logic.

**splitRight(groupId) function:** calls layout.addGroup(); loads active tab of source group into new group if active tab exists (duplicate per CONTEXT.md: "Duplicate objects are allowed").

**getActiveEditorArea() function:** returns `document.getElementById('editor-area-' + layout.activeGroupId)`. Exported as `window.getActiveEditorArea`.

**initWorkspaceLayout() function:** migrateTabState(); restore or create new layout with single group-1; recreateGroupSplit(layout); wire click events on each group container to setActiveGroup on focus.

Expose on window:
- `window._workspaceLayout = layout`
- `window.getActiveEditorArea = getActiveEditorArea`
- `window.splitRight = splitRight`
- `window.setActiveGroup = setActiveGroup`
- `window.initWorkspaceLayout = initWorkspaceLayout`
