This is the read-only face of the object tab. It displays:

a) **Property table** (two-column grid, GitHub sidebar style):
```html
<div class="object-read-view">
  <div class="property-table">
    {% for prop in form.properties %}
      {% set vals = values.get(prop.path, []) %}
      {% if vals %}
      <div class="property-row">
        <div class="property-label">{{ prop.name }}</div>
        <div class="property-value">
          {% if prop.target_class %}
            {# Reference property: render as pills #}
            {% for v in vals %}
            <span class="ref-pill"
                  onclick="openTab('{{ v }}', '{{ ref_labels.get(v, v) }}')"
                  title="{{ v }}">
              <span class="ref-pill-dot"></span>
              {{ ref_labels.get(v, v) }}
            </span>
            {% endfor %}
          {% elif prop.datatype == 'http://www.w3.org/2001/XMLSchema#date' or prop.datatype == 'http://www.w3.org/2001/XMLSchema#dateTime' %}
            {% for v in vals %}{{ v | format_date }}{% if not loop.last %}, {% endif %}{% endfor %}
          {% elif prop.datatype == 'http://www.w3.org/2001/XMLSchema#boolean' %}
            {% for v in vals %}
              {% if v == 'true' %}<span class="bool-icon bool-true" title="True">&#10003;</span>{% else %}<span class="bool-icon bool-false" title="False">&#10007;</span>{% endif %}
            {% endfor %}
          {% elif prop.datatype == 'http://www.w3.org/2001/XMLSchema#anyURI' %}
            {% for v in vals %}<a href="{{ v }}" target="_blank" rel="noopener" class="uri-link">{{ v }}</a>{% if not loop.last %}<br>{% endif %}{% endfor %}
          {% else %}
            {% for v in vals %}{{ v }}{% if not loop.last %}, {% endif %}{% endfor %}
          {% endif %}
        </div>
      </div>
      {% endif %}
    {% endfor %}
  </div>

  {# Horizontal rule separator #}
  {% if body_text %}
  <hr class="body-separator">
  {# Markdown body rendered area #}
  <script type="text/plain" id="md-source-{{ safe_id }}">{{ body_text }}</script>
  <div class="markdown-body" id="md-rendered-{{ safe_id }}">
    <div class="markdown-loading">Rendering...</div>
  </div>
  <script>
    (function() {
      if (typeof window.renderMarkdownBody === 'function') {
        window.renderMarkdownBody('md-source-{{ safe_id }}', 'md-rendered-{{ safe_id }}');
      } else {
        // Fallback: show raw text
        var src = document.getElementById('md-source-{{ safe_id }}');
        var tgt = document.getElementById('md-rendered-{{ safe_id }}');
        if (src && tgt) { tgt.textContent = src.textContent; }
      }
    })();
  </script>
  {% endif %}
</div>
```

Handle edge cases:
- If `form` is None, show "No schema available" message
- If `values` is empty and no `body_text`, show "This object has no content yet" message
- The `safe_id` variable comes from the parent template (object_tab.html)

c) The inline `<script>` for editor initialization must be DEFERRED. Do NOT initialize CodeMirror and Split.js on load. Instead, wrap them in a function `window._initEditMode_{{ safe_id }}` that gets called only when the user first switches to edit mode. Store the flag in a data attribute `data-edit-initialized="false"` on the flip container. This prevents Pitfall 3 (expensive resources initialized upfront for read-only viewing).

The script becomes:
```html
<script>
(function() {
  // Store initialization function for deferred edit mode setup
  window['_initEditMode_' + '{{ safe_id }}'] = function() {
    var container = document.getElementById('flip-{{ safe_id }}');
    if (!container || container.dataset.editInitialized === 'true') return;
    container.dataset.editInitialized = 'true';

    // ... existing editor init code (Promise.race, skeleton, fallback, Split.js) ...
  };

  // If mode is 'edit' (newly created objects), initialize immediately
  {% if mode == 'edit' %}
  var flipInner = document.getElementById('flip-inner-{{ safe_id }}');
  if (flipInner) flipInner.classList.add('flipped');
  var toggleBtn = document.getElementById('mode-toggle-{{ safe_id }}');
  if (toggleBtn) { toggleBtn.textContent = 'Done'; }
  var saveBtn = document.getElementById('save-btn-{{ safe_id }}');
  if (saveBtn) saveBtn.style.display = '';
  window['_initEditMode_' + '{{ safe_id }}']();
  {% endif %}
})();
</script>
```

d) Keep the existing editor initialization logic (Promise.race with timeout, skeleton, fallback) but move it INSIDE the `_initEditMode_` function.
