<task id="27-01-4">
  <title>Create backend/app/vfs/auth.py — SemPKMTokenAuthenticator</title>
  <action>
Create `backend/app/vfs/` package (add `__init__.py`) and `backend/app/vfs/auth.py` with the custom wsgidav HTTPAuthenticator.

The authenticator validates Basic auth credentials against the `api_tokens` table. Because wsgidav runs in a WSGI thread pool (not the asyncio event loop), the auth check must be fully synchronous.

**Implementation approach:**

```python
"""SemPKM token authenticator for wsgidav.

Validates HTTP Basic auth credentials (username=email, password=API token)
against the api_tokens table using a synchronous SQLAlchemy engine.

wsgidav's WSGI thread pool cannot use asyncio, so this uses a dedicated
sync engine with connection pooling (StaticPool for SQLite thread safety).
"""

import hashlib
from sqlalchemy import create_engine, select
from sqlalchemy.orm import Session
from wsgidav.dc.base_dc import BaseDomainController

from app.auth.models import User, APIToken
from app.config import settings


class SemPKMTokenAuthenticator(BaseDomainController):
    """Validates Basic auth (email + API token) for WebDAV access."""

    def __init__(self, wsgidav_app, config):
        super().__init__(wsgidav_app, config)
        db_url = settings.database_url
        # Use StaticPool for SQLite to ensure thread-safe single connection
        if "sqlite" in db_url:
            from sqlalchemy.pool import StaticPool
            self._engine = create_engine(
                db_url.replace("sqlite+aiosqlite", "sqlite"),
                connect_args={"check_same_thread": False},
                poolclass=StaticPool,
            )
        else:
            self._engine = create_engine(
                db_url.replace("+asyncpg", "").replace("+aiosqlite", "")
            )

    def get_domain_realm(self, path_info, environ):
        return "SemPKM"

    def require_authentication(self, realm, environ):
        return True

    def is_share_anonymous(self, path_info):
        return False

    def basic_auth_user(self, realm, user_name, password, environ):
        """Validate username (email) + password (API token).

        Returns True if valid, False otherwise.
        Also sets environ["sempkm.user_id"] for use by the DAV provider.
        """
        token_hash = hashlib.sha256(password.encode()).hexdigest()
        with Session(self._engine) as session:
            user = session.execute(
                select(User).where(User.email == user_name)
            ).scalar_one_or_none()
            if user is None:
                return False
            token_row = session.execute(
                select(APIToken).where(
                    APIToken.user_id == user.id,
                    APIToken.token_hash == token_hash,
                )
            ).scalar_one_or_none()
            if token_row is None:
                return False
            environ["sempkm.user_id"] = str(user.id)
            environ["sempkm.user_email"] = user.email
            return True

    def supports_http_digest_auth(self):
        return False
```

Note: The `database_url` in config uses `sqlite+aiosqlite:///...` for async. Strip the async driver prefix (`aiosqlite` → plain `sqlite`, `asyncpg` → `postgresql`) to get a sync-compatible URL.
