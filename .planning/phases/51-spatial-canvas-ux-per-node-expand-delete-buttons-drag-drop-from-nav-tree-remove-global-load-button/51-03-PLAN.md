  **service.py — add 4 new methods to CanvasService:**

  1. `async def list_sessions(self, user_id, db) -> list[dict]`:
     - Read the `canvas.sessions.index` UserSetting key.
     - Parse JSON value. Return empty list if not found.
     - Each entry has `{id, name, updated_at}`.

  2. `async def save_session_as(self, user_id, name, document, db) -> str`:
     - Generate `session_id = str(uuid.uuid4())[:8]`.
     - Save document via `self.save_document(user_id, session_id, document, db)` (this also commits).
     - Actually, avoid double-commit: call `_upsert_setting` for doc and meta keys directly without committing, then update the index, then commit once.
     - Read current sessions list via `list_sessions()`.
     - Append `{"id": session_id, "name": name, "updated_at": datetime.now(tz=timezone.utc).isoformat()}`.
     - Upsert `canvas.sessions.index` with updated JSON.
     - Set active session: upsert `canvas.active_session` to `session_id`.
     - Commit.
     - Return `session_id`.

  3. `async def get_active_session_id(self, user_id, db) -> str | None`:
     - Read `canvas.active_session` UserSetting key.
     - Return value or None.

  4. `async def set_active_session(self, user_id, session_id, db) -> None`:
     - Upsert `canvas.active_session` to `session_id`.
     - Commit.

  5. `async def delete_session(self, user_id, session_id, db) -> bool`:
     - Read sessions index, find entry with matching id.
     - If not found, return False.
     - Remove from list, upsert index.
     - Delete the document and meta UserSetting rows for this session_id (use `db.execute(delete(...))` or just set to empty).
     - If the active session was deleted, clear `canvas.active_session` or set to another session if available.
     - Commit.
     - Return True.

  6. `async def migrate_default_canvas(self, user_id, db) -> str | None`:
     - Check if `canvas.sessions.index` exists. If yes, return None (already migrated).
     - Check if `canvas.default.document` exists and has content.
     - If yes: create a session named "My Canvas" using `save_session_as` with the default document content.
     - Return the new session_id.
     - If no default canvas: return None.

  **router.py — add 3 new endpoints** (IMPORTANT: place these BEFORE the `/{canvas_id}` routes so they don't get caught by the path parameter):

  1. `GET /api/canvas/sessions/list` — returns `SessionListResponse`:
     - Call `service.migrate_default_canvas(user.id, db)` first (one-time migration).
     - Call `service.list_sessions(user.id, db)`.
     - Call `service.get_active_session_id(user.id, db)`.
     - Return `SessionListResponse(sessions=sessions, active_session_id=active)`.

  2. `POST /api/canvas/sessions` — creates a new named session:
     - Accept `SessionCreateBody`.
     - Call `service.save_session_as(user.id, body.name, body.document, db)`.
     - Return `{"session_id": session_id, "name": body.name}`.

  3. `DELETE /api/canvas/sessions/{session_id}` — deletes a session:
     - Call `service.delete_session(user.id, session_id, db)`.
     - If False, raise 404.
     - Return `{"deleted": True}`.

  4. `PUT /api/canvas/sessions/{session_id}/activate` — set active session:
     - Call `service.set_active_session(user.id, session_id, db)`.
     - Return `{"active_session_id": session_id}`.
