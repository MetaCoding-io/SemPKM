  3. **Modify `renderNodes()`** — change the node HTML (lines 218-229) to:
     ```

     </article>
     ```

  4. **Add hint visibility toggle** at the end of `renderNodes()`:

  5. **Extend `onLayerClick()`** — replace the `.spatial-node-toggle` handler with three new handlers using `event.target.closest()`:

     a. `.spatial-node-chevron` click: Toggle `model.collapsed`, call `renderNodes()`.

     b. `.spatial-node-delete` click: Call `removeNode(nodeId)`.

     c. `.spatial-node-expand` click: Call `toggleExpand(nodeId)`.

     Add early returns after each handler to prevent event propagation to panning.

  6. **Implement `removeNode(nodeId)`**:
     - Filter node from `state.nodes`
     - Filter edges referencing that node from `state.edges`
     - Clean up provenance: remove nodeId from any expand's child list in `state.expandProvenance`, and delete `state.expandProvenance[nodeId]` if it existed
     - Call `renderNodes()`

  7. **Implement `toggleExpand(nodeId)`**:
     - If `state.expandProvenance[nodeId]` exists → **collapse**: filter out nodes that are exclusively owned by this expand (check no other provenance references them), remove their edges, delete `state.expandProvenance[nodeId]`, call `renderNodes()`, `setStatus('Collapsed neighbors')`.
     - If no provenance → **expand**: call `expandNode(nodeId)`.

  8. **Implement `async expandNode(nodeId)`**:
     - Find the node model via `findNode(nodeId)`.
     - Fetch `/api/canvas/subgraph?root_uri={encoded_uri}&depth=1`.
     - Build `existingIds` map from current `state.nodes`.
     - For each new node in response: position in a circle around the expanded node (radius 200, angle = idx/total * 2*PI). Push to `state.nodes`.
     - Collect `newNodeIds` array.
     - Merge edges (reuse the dedup pattern from `mergeSubgraph` — build `existingEdgeIds` set, skip duplicates).
     - Set `state.expandProvenance[nodeId] = newNodeIds`.
     - Call `renderNodes()`, `setStatus('Expanded ' + newNodeIds.length + ' neighbors')`.
     - Wrap in try/catch, on error: `setStatus('Expand failed', true)`.

  9. **Extend `getDocument()`** to include `expandProvenance: state.expandProvenance` in the returned object.

  10. **Extend `applyDocument()`** to restore `state.expandProvenance = document.expandProvenance || {}`.
