**1. Create `frontend/static/js/named-layouts.js`** — a self-contained IIFE that manages named workspace layouts in localStorage. Expose API on `window.SemPKMLayouts`.

Keys:
- `sempkm_layouts` — JSON object: `{ "Layout Name": { layout: <SerializedDockview>, savedAt: <ISO string> } }`
- `sempkm_layout_current` — auto-save: raw `JSON.stringify(dv.toJSON())` value

Functions to expose:
- `save(name)` — captures `window._dockview.toJSON()`, stores under name in `sempkm_layouts`. If name already exists, silently overwrites (per discretion: silent replace with toast handled by caller). Returns `true` on success.
- `restore(name)` — reads named layout from `sempkm_layouts`, calls `window._dockview.fromJSON(entry.layout)`. Wraps in try/catch. Returns `{ success: boolean, skipped: string[] }`. On catch, compare `dv.panels` against saved panel list to build skipped array.
- `remove(name)` — deletes key from `sempkm_layouts`. Returns `true` if existed and was deleted.
- `list()` — returns array of `{ name: string, savedAt: string }` sorted by savedAt descending (most recent first).
- `autoSave()` — writes `window._dockview.toJSON()` to `sempkm_layout_current`. Called by workspace-layout.js onDidLayoutChange.
- `restoreCurrent()` — reads `sempkm_layout_current`, parses JSON, returns the object or null.

Internal helpers:
- `_readLayouts()` — `JSON.parse(localStorage.getItem('sempkm_layouts')) || {}`
- `_writeLayouts(obj)` — `localStorage.setItem('sempkm_layouts', JSON.stringify(obj))`
- `_escapeHtml(str)` — escapes `<`, `>`, `&`, `"` for safe display in ninja-keys (prevent XSS from layout names containing HTML)

b) **Auto-save handler** (line 143-146): Replace `sessionStorage.setItem(DV_LAYOUT_KEY, ...)` with `localStorage.setItem('sempkm_layout_current', JSON.stringify(dv.toJSON()))`. Keep the htmx re-process logic unchanged.

c) **Init restore** (lines 179-196): Implement migration logic:
   - First try `localStorage.getItem('sempkm_layout_current')`
   - If not found, fall back to `sessionStorage.getItem('sempkm_workspace_layout_dv')`
   - If found in sessionStorage, migrate to localStorage and clear sessionStorage key
   - Parse and pass to `dv.fromJSON()` as before
