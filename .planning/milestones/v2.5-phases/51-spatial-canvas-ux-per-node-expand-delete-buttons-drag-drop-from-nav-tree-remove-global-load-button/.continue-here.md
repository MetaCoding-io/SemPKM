### 55ffc23 ŌĆö WIP: drag-drop + body fetch fixes

<decisions_made>

- **Side-channel pattern for dockview**: Store drag payload in `window.__canvasDragPayload` and use `dragend` as fallback when `drop` never fires
- **Capture-phase listeners on document**: Canvas drag handlers on `document` with `capture: true` using `isOverCanvas()` bounding-rect check
- **Model-agnostic body fetch**: SPARQL `FILTER(STRENDS(STR(?p), ":body"))` matches any model's body predicate
- **Never clear lastDragOverCanvas in dragleave**: Dockview overlays cause spurious events; only clear in drop/dragend
- **Single edge SVG pass**: Only the edgePoint()-based pass draws edges; removed naive center-to-center first pass

</decisions_made>
