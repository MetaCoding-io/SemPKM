About SemPKM

SemPKM is a semantic personal knowledge management system built on event-sourced RDF. It stores knowledge as linked data triples in an RDF4J triplestore, using an immutable event log for all writes and a materialized current state graph for reads.

Use this dev console to interact with the system: run SPARQL queries against the knowledge graph, create and modify objects via commands, and monitor system health.
