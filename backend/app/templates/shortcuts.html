Keyboard Shortcuts

These shortcuts use the Alt key to avoid browser shortcut conflicts. On macOS, use Option.

General

Alt + K  /  F1 Open command palette
Alt + , Open settings
Alt + [ Toggle explorer panel
Alt + ] Toggle details panel
Alt + J Toggle bottom panel
Alt + \ Split editor right
Alt + 1–4 Focus editor group

Objects

Alt + N Create new object
Alt + S Save current object
Alt + E Toggle read / edit mode
Alt + W Close current tab

Tools

Alt + Shift + V Run validation
