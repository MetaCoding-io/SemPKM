Keyboard Shortcuts

These shortcuts are available throughout the application. On macOS, use Cmd instead of Ctrl.

General

Ctrl + K Open command palette
Ctrl + \ Toggle sidebar
Ctrl + [ Toggle explorer panel
Ctrl + ] Toggle details panel

Objects

Ctrl + N Create new object
Ctrl + S Save current object
Ctrl + E Toggle read / edit mode
Ctrl + W Close current tab

Tools

Ctrl + Shift + V Validate current object
