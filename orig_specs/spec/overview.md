# SemPKM Spec Overview

This directory contains the normative-ish specifications for SemPKM v1:

- Mental Model bundle format and manifest schema
- View and Dashboard specifications
- Projection and sidecar schemas
- SHACL UI profile used to generate forms and linting UX
- Command API (write surface)

The specs are designed to be:
- easy to implement in a web-first, local-first product
- compatible with marketplace distribution (Mental Models)
- stable enough for long-term evolution (v2+ migrations are explicitly future work)
