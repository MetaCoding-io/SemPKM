# Decision Log — v0.3

## Confirmed decisions
- Event log is canonical truth (event-sourced)
- Edges are first-class resources; triple-term/RDF-star is projection/interchange
- SHACL drives forms + linting (assistive, async validation)
- Dashboards are distinct from object rendering; parameterized; registry supported
- View/dash IDs are namespaced by modelId; cross-model embedding is export-only, private-by-default
- Projections are read-only v1; IRI-path mapping; sidecars for meta/edges; per-object only
- API is SPARQL for reads + minimal command API for writes
- Webhooks are simple v1; advanced delivery/security deferred to v3
