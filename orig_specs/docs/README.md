# SemPKM

SemPKM is a semantics-native personal knowledge management platform.

- **RDF + SPARQL** for querying (reads)
- **Event-sourced command API** for writes
- **SHACL-driven UI** for forms and linting
- **Mental Models** (ontology + shapes + views + projections) for instant “PKM experiences”
- **Read-only filesystem projections** for interoperability with tools like Obsidian
